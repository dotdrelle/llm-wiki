# Objectif du pack Notes projet

## Finalité

Le pack Notes projet sert à maintenir une documentation vivante et régulièrement actualisée.

Contrairement au pack REAS, il ne cherche pas à produire un dossier formel complet. Il produit des notes ciblées, lisibles et faciles à régénérer.

## Problématique couverte

Le pack répond à la même problématique que le pack REAS :

- cadrer un besoin ;
- identifier les exigences ;
- décrire les options d’architecture ;
- comparer les options ;
- tracer les décisions ;
- préparer l’exploitation.

## Différence principale

Une note doit être courte, maintenable et centrée sur un objet documentaire.

Elle doit permettre à une équipe projet de retrouver rapidement l’état courant du sujet.
