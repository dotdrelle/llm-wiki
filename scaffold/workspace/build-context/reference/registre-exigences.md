# Registre des exigences pour notes projet

## Rôle

Le registre des exigences est la source de vérité des exigences dans le format Notes projet.

## Format

| ID | Domaine | Exigence | Priorité | Statut | Source | Commentaire |
|---|---|---|---|---|---|---|

## Domaines

- Fonctionnel
- Données
- Interface
- Sécurité
- Exploitation
- Performance
- Résilience
- Réglementaire
- Organisationnel
- Technique

## Statuts

- Confirmée
- Déduite des sources
- À confirmer
- Non couverte
- Contradictoire
- Absente des sources

## Règle

Les autres notes doivent référencer les exigences par ID.
